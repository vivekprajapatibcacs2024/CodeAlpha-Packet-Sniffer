CodeAlpha Internship Task 1
**Network Packet Sniffer using Python**

1. Project Overview
This project was developed as part of the CodeAlpha Cyber Security Internship.
The objective of this task is to design and implement a basic network packet sniffer capable of capturing and analyzing live network traffic in real time.
The application monitors packets flowing through the network and extracts essential information such as IP addresses, protocols, ports, and DNS queries.

2. Objectives

Capture live network packets
Identify protocol types (ICMP, TCP, UDP)
Display source and destination IP addresses
Detect DNS queries
Analyze packet length
Understand real-time traffic behavior

3. Technologies Used
Python 3
Scapy Library
Kali Linux
Nmap (for network host discovery)

4. Features
Real-time packet capturing
Protocol identification
IP address extraction
Port number detection
DNS query monitoring (UDP Port 53)
ICMP traffic detection (Ping analysis)
Lightweight and terminal-based execution

5. Working Principle
The packet sniffer uses the Scapy library to capture network packets.
When a packet is detected, the program:
Extracts the IP layer
Identifies the protocol type
Displays source and destination addresses
Checks for TCP or UDP ports
Detects DNS query information
Prints packet length
The program continuously monitors traffic until manually stopped.

6. Execution Steps
6.1 Running the Sniffer
   
sudo python3 sniffer.py
The program begins monitoring live network traffic.

6.2 ICMP Traffic Testing

ping google.com
The sniffer captures ICMP packets generated by the ping command.

6.3 DNS Query Testing

nslookup google.com
The sniffer detects DNS requests sent over UDP port 53 and displays the queried domain name.

6.4 Network Host Discovery (Additional Analysis)

sudo nmap -sn 192.168.31.0/24
This command identifies active devices on the local network.
